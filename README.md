## **部署条件**
1、PHP版本不能小于5.3版本

## **支持的服务器和数据库环境**

支持Windows/Unix服务器环境
可运行于包括Apache、IIS和nginx在内的多种WEB服务器和模式
框架默认只支持Mysql数据库后期可扩展
